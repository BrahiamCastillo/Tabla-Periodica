Tabla periódica dinámica, esta se genera automáticamente recibiendo datos a travéz de un array asociativo,
dicho array se divide en elementos y clases, la cual esta última contiene clases de CSS para añadirle el 
respectivo color a cada elemento, mientras que la parte "elemento" contiene la simbología de cada elemen-
to añadido en la tabla periódica, cuando la parte "elemento" recibe un valor null o vacío, la celda en don-
de se carga dicho elemento adopta un color gris, dando a entender de que esta es inválida.
